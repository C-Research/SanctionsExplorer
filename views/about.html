About OFACasaurus

OFACasaurus reimagines the OFAC SDN List website, a list of individuals, entities, and vehicles that are sanctioned by the Treasury Department. This list is notoriously difficult to search yet plays a crucial role in foreign investigations. We've redesigned the experience of searching the database, providing access to all of the same data with a more seamless interface and additional features to streamline the investigative process.

OFACasaurus was created by Archer (a Berkeley-based nonprofit using tech to improve human rights and human security) in collaboration with C4ADS (a nonprofit data-driven think tank in D.C.).

Features

Better search capabilities

Unsure how a name transliterates? Are multiple similar spellings available? With fuzzy/inexact matching, items that might have previously gone unfound will no longer slip through the cracks. Additionally, there are more filters available to allow for searching by other categories, so it's easier to find what you're looking for.

Press release scraping

The Treasury releases additional context when they update the list. These releases are indexed & searchable via the OFACasaurus platform. These press releases can even be used to search for entities who were on the SDN list at one point but were removed, since their identifying information will have been released in a press release.

Bulk searching

Rather than manually searching dozens of names, simply upload a text file containing identifying information. We'll generate a PDF report that includes data on every entity listed in the document.

Printable reports

Even regular search queries can be printed -- simply run the print keyboard shortcut (control + P on Windows; command + P on Mac) and all of the cards on your screen will be printed. Expand the entities of interest and their details will be included in the printout!

There are plenty of other ways that OFACasaurus stands out from the Treasury's website, including an interface that's modern, updated, and requires fewer clicks to navigate. If you'd like to see a feature added or report a bug, please email ofacasaurus@archerimpact.com.
